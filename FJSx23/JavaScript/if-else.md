### Syntax


### Ternary Operator:
Shorthand for writing if statements:

```javascript
let age = 20;

//                (condition) ?   (run if true)  :   (run if false)    ;
const oldEnough = (age >= 21) ? "You may enter." : "You may not enter.";

oldEnough;
```
